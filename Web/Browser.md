## Debounce & Throttle

### Debounce

개념

- 연속된 이벤트에 대해 마지막 이벤트만 처리하는 방식
- 연속된 이벤트가 종료되고 일정 시간 후 한번만 실행

사용 사례

- 검색창 자동완성
- 입력 폼 validation
- 윈도우 resize 이벤트
- 버튼 중복 클릭 방지

### Throttle

개념

- 연속된 이벤트를 주기적으로 실행하는 방식
- 이벤트 발생 중에도 주기적으로 실행

사용 사례

- 무한 스크롤
- 버튼 연타 방지
- 마우스 이동 추적
- 이벤트가 진행 중일 때에도 주기적으로 체크가 필요할 때
